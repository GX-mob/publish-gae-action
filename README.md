# publish-gae-action

This action deploying your application to Google App Engine.

## Example

[BoardCAM.org](https://github.com/BoardCAM/BoardCAM.org/blob/master/.github/workflows/publishgae.yml)

## Usage
```yaml
steps:
- name: Publish to GAE
  uses: zxyle/publish-gae-action@master
```
